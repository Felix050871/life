Generazione Turni da Coperture
Come funziona:
  1. Coperture: Utilizza le coperture create per definire orari e ruoli
  2. Periodo: Seleziona il periodo per cui generare i turni
  3. Assegnazione: Il sistema assegna automaticamente gli utenti
  4. Bilanciamento: Distribuzione equa del carico di lavoro
Caratteristiche:
  • Basato sulle coperture create per ogni sede
  • Rispetta le richieste di ferie approvate
  • Considera le percentuali part-time
  • Assegna solo utenti con ruoli appropriati
  • Evita sovrapposizioni negli orari
Note Importanti
Prima di generare turni:
  • Assicurati che gli utenti della sede siano attivi e configurati correttamente
  • Verifica che le percentuali part-time siano impostate
  • Controlla che non ci siano richieste di ferie in conflitto
    • {% if is_admin %}
  • Configura le coperture di reperibilità prima di generare i relativi turni
    • {% endif %}
